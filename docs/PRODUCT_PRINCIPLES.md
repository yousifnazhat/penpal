# PenPal Product Principles

## What PenPal Is

PenPal is a professional enumeration companion for authorized pentesters.

It helps with:

- organizing targets
- planning checks
- parsing outputs
- extracting missed evidence
- connecting evidence into likely paths
- generating exact syntax
- preparing notes and reports

## What PenPal Is Not

PenPal is not an auto-exploitation tool.

It should not hide commands, invent findings, or encourage unsafe actions. It should make the operator faster while keeping them in control.

## Product Promise

PenPal should reduce cognitive load without reducing operator understanding.

The user should feel:

- "I know what I have tried."
- "I know why this next check matters."
- "I can copy the correct syntax."
- "I can trace every suggestion back to evidence."
- "I can turn this into a clean report."

## Quality Bar

Features should be:

- explainable
- testable
- auditable
- safe by default
- useful from the CLI
- consumable by the future frontend

If a feature only looks impressive but does not improve enumeration clarity, it should wait.
